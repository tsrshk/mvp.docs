# ADR-008 · Organization-as-tenant; multi-tenant-ready, single-tenant-first

**Status:** accepted · **Date:** 2026-07-02 (pre-existing)

## Context
We need to expand into Phase 2 without rewriting the data model.

## Decision
Hierarchy `organization → subdivision → membership(user↔subdivision+role)`; `users` is global. `organization_id` is denormalized onto every operational row (`ondelete=RESTRICT`); tenant repositories require a scope in the constructor; the scope is taken only from a signed JWT. Phase 1 uses a single default tenant.

## Consequences
Hard isolation from day one, with a minimal implementation.
