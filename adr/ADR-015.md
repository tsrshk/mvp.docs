# ADR-015 · Documentation regime (trust order, archive is inert)

**Status:** accepted · **Date:** 2026-07-02

## Context
Sedimentary layers of pivots, conflicting documentation, the agent reading stale material.

## Decision
Documentation is maintained as a local Confluence-lite: canonical live documents + `work/` + `archive/`, front-matter with version/date/status, an explicit trust order (**code > normative > descriptive > product > work > archive**); the agent does not read the archive.

## Consequences
This file and all of `mvp.docs/` are subject to the regime (implemented as an Obsidian/Jira vault: folders-by-type + wikilinks).
