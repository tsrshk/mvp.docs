# ADR-001 · LCOS — точка ввода накладных, а не POS

**Status:** accepted · **Date:** 2026-07-02 (pre-existing)

## Контекст
Есть соблазн расшириться в операционный учёт / POS.

## Решение
LCOS собирает и анализирует данные поверх ERP; это **не** POS и **не** операционный учёт. Интеграция с планшетом/POS и бухгалтерия остаются в ERP (Esupl). Конкретно, система является точкой записи накладных: фото → OCR → сопоставление с SKU → валидация → payload для Esupl → локальное сохранение + (за гейтом) запись в POS.

## Последствия
Расчёт зарплат, смены, финансовая отчётность — вне области. Граница продукта ясна.
