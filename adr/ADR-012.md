# ADR-012 · The frontend stores no secrets

**Status:** accepted · **Date:** 2026-07-02 (pre-existing)

## Context
XSS token theft, keys in the browser.

## Decision
Auth only via HttpOnly cookies; all secrets (LLM keys, ERP tokens) live on the backend, encrypted, behind a write-only / read-masked API. `VITE_*` covers only non-secret endpoint/provider toggles. The live provider paths are only `backend` / `mock`.

## Consequences
Browser-direct LLM/ERP is removed as dead code (BACKLOG A2).
