# ADR-012 · Фронтенд не хранит секретов

**Status:** accepted · **Date:** 2026-07-02 (pre-existing)

## Контекст
Кража токена через XSS, ключи в браузере.

## Решение
Аутентификация только через HttpOnly cookies; все секреты (LLM-ключи, ERP-токены) живут на бэкенде, зашифрованы, за write-only / read-masked API. `VITE_*` покрывает только несекретные тумблеры endpoint/provider. Живые провайдер-пути — только `backend` / `mock`.

## Последствия
Browser-direct LLM/ERP удаляется как мёртвый код (BACKLOG A2).
