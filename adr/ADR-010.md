# ADR-010 · Secret encryption at-rest (Fernet envelope, versioned KEK)

**Status:** accepted · **Date:** 2026-07-02 (pre-existing)

## Контекст
Секреты, хранимые в DB в открытом виде = утечка при любом дампе.

## Решение
Fernet, формат `enc:v2:<key_id>:<token>`; KEK(и) живут только в `.env` (`SECRETS_ENC_KEY` + `_KEY_ID` + `_KEYS_OLD`); ротация через продвижение нового KEK; `validate_keyring()` при старте.

## Последствия
Дамп DB без `.env` бесполезен; ротация без потери старых шифртекстов.
