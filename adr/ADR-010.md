# ADR-010 · Secret encryption at-rest (Fernet envelope, versioned KEK)

**Status:** accepted · **Date:** 2026-07-02 (pre-existing)

## Context
Secrets stored in the DB in plaintext = a leak on any dump.

## Decision
Fernet, format `enc:v2:<key_id>:<token>`; KEK(s) live only in `.env` (`SECRETS_ENC_KEY` + `_KEY_ID` + `_KEYS_OLD`); rotation by promoting a new KEK; `validate_keyring()` at startup.

## Consequences
A DB dump without `.env` is useless; rotation without losing old ciphertexts.
